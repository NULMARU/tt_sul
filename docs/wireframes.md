# 5축 메인 화면 + Daily Story 와이어프레임

> 표기: `[버튼]`, `┌─┐` 패널, `●` 진행 점, `🟢🟡🔴` 난이도, `▸` 진입 액션

---

## 0. 화면 지도 (라우트)

```
/                     홈 (지금 추천 + 회로 다이얼 + Daily Story + 5축 칩)
/now                  맥락 자동 진입 (시간∩장소∩요일 교집합)
/axis/day             📅 일자 30 그리드
/axis/stage           🪜 단계 트리
/axis/place           🗺️ 집·도시 미니맵
/axis/situation       🎬 씬 라이브러리
/axis/time            ⏰ 24h 클럭
/lesson/:id           레슨 (회로 5스텝: U→A→R→P→I)
/story/:id            오늘의 스토리 (Daily Story)
/quiz/:id
/review               SRS 큐 (n=3 / wrong=1 옵션)
/memory-map           Phrase Memory Map (망각 시각화)
/journal              일기 + 다음날 퀴즈 변환
/toolbelt             설정·내보내기·녹음·음성
```

│                   ╱     ╲                       │   외곽 4개 = 단일 모드

│                                                 │
│ ─── 다른 축으로 탐색 ───                         │

**상호작용**
- Now 카드는 5초 후 페이드 인 (살짝 들어오게)

- 5축 칩 가로 스와이프 가능

## 7. 회로 다이얼 진입 시 — `/lesson/:id` (5스텝 흐름)

```
스텝 진행 바: 🧠──👂──📖──🗣️──🔁
              1    2    3    4    5

스텝1 🧠 Understand
  hook → analogy → highlight 카드 (좌우 스와이프, 자동 TTS)

스텝2 👂 Absorb
  표현 카드 N개 · 받아쓰기 모드 · 쉐도잉(녹음+TTS비교)

스텝3 📖 Read (Daily Story가 그 레슨 표현 포함이면 자동)
  난이도 토글 🟢🟡🔴 · 학습 표현 하이라이트 · 미니 이해 퀴즈 3

스텝4 🗣️ Produce
  3단계 누적 작문 (동작→상태→이유) · 자유 작문 → LLM 채점

스텝5 🔁 Imprint
  6종 혼합 퀴즈 → SRS 큐 등록 → "내 일기 한 문장"
```

상단에 컨텍스트 칩이 항상 보임:
`📅 Day 1 · 🪜 Stage 1 · 🗺️ 집 침실 · 🎬 기상 · ⏰ 아침`

## 9. 🗺️ Phrase Memory Map (`/memory-map`)

```
┌─────────────────────────────────────────────────┐
│  내 기억 지도          정렬: [강도▼] [장소] [시간]│
├─────────────────────────────────────────────────┤
│ ▣ wake up      ▩ snooze    ▤ rub eyes    ▣ yawn │  ▣=강함 ▩=중간
│ ▣ get out      ▤ stretch   ░ throw off    ▤ sit │  ▤=약함 ░=망각
│ ░ wash face    ░ sunscreen  ▩ comb hair   ▤ get │
│ ...                                              │
│                                                  │
│ ─── 흐려진 12개를 지금 복습?  [▸ 1분만]          │
└─────────────────────────────────────────────────┘
```

색은 시간 경과·정답률에 따라 자동 페이드. 탭 → 그 표현 카드 → 회로 미니 1스텝.
